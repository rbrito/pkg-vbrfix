2.3 Compilation and installation

In order to compile and install Vbrfixc on your system, type the following in the base directory of the Vbrfixc distribution:

% ./configure
% make
% make install

Since Vbrfixc uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at William Pye
